# NPower JDA Cohort 2026

Welcome to the NPower JDA Cohort 2026 GitHub Organization!

Our goal is to collaborate, learn, build projects together, and gain real-world experience using GitHub workflows and best practices.

---

## 🎯 Purpose

This organization provides a shared space where cohort members can:

- Collaborate on projects
- Practice professional Git workflows
- Review each other's code
- Build stronger portfolios
- Share ideas and resources
- Gain experience working in development teams

By contributing to organization repositories, members can showcase teamwork and collaboration on their GitHub profiles.

---

## 📁 Repository Ideas

Members are encouraged to create and contribute to repositories such as:

- `cohort-projects`

- `community-projects`

- Python Applications

- Learning & Practice Repositories

We encourage everyone to use GitHub's collaboration features.

Use Issues to:

- Propose ideas

- Suggest improvements
- Assign tasks

### Pull Requests

Use Pull Requests to:

- Submit changes
- Request feedback
- Practice code reviews
- Discuss implementation details

### Projects

Use Project Boards to:

- Track progress
- Organize tasks
- Plan milestones

### Discussions

Use Discussions to:

- Brainstorm ideas
- Ask questions
- Share resources
- Coordinate project activities

## 📊 Contribution Tracking

GitHub automatically tracks:

- Number of commits
- Pull Requests opened and merged
- Issues created
- Code Reviews performed
- Repository contributions

These contributions help demonstrate collaboration, consistency, and technical engagement.

Everyone's GitHub profile will reflect activity within the organization.

---

## 📋 Basic Rules

1. No direct commits to the `main` branch.
2. Create a separate branch for each feature or fix.
3. Open a Pull Request for all changes.
4. Have another member review your Pull Request before merging whenever possible.
5. Create or reference an Issue before starting significant work.
6. Be respectful and constructive during reviews and discussions.
